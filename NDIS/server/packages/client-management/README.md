# Client Management Module

NDIS Client Management API

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Install type definitions:
   ```bash
   npm install --save-dev typescript @types/node @types/express
   ```
3. Build TypeScript:
   ```bash
   npx tsc
   ```
4. Start server:
   ```bash
   node dist/index.js
   ```

## Endpoints
- `/api/clients` CRUD operations
